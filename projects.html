Accessibility Projects

Here are some of the accessibility-focused projects I've been working on:

Virtual Taylor Frame

An interactive grid application designed to simulate a physical Taylor Frame, commonly used in mathematics education for visually impaired students. This digital version provides an accessible experience with keyboard navigation, text-to-speech feedback, and screen reader compatibility.

Features:

  • Interactive 18x25 grid with customizable size
  • Keyboard navigation and shortcuts
  • Text-to-speech feedback with cytolk
  • Sound effects for different actions
  • Auto-shift, smart delete, and fast move options
  • Full screen reader compatibility

The Internet Analog Radio

An audio-first experience that brings the nostalgia of analog radio to the internet age. This project focuses on creating an accessible, sound-based interface for exploring online audio content.

Features:

  • Accessible audio-first interface
  • Python-based implementation
  • Built with accessibility in mind
